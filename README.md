# Kodana - コード読む力を育てる無料エディタ

## 概要
Kodana（コダナ）は、**コードを読む・書く・試す** を楽しく学べる無料ツールです。  
- 読むパート：解説を読むだけ  
- 書くパート：入力して判定  
- ドリル/クイズ：実践問題  
子供から大人まで、プログラミングの基礎を身につけよう！

**Windows専用（.exeポータブル）**  
解凍不要！ ダブルクリックで起動します。

## 使い方（クイックスタート）
1. zipを解凍  
2. `kodana.exe` または `kodanapen.exe` をダブルクリック  
3. 「コダナの使い方フォルダ」の中のサンプル教材を読み込んでみよう！  
   - メタ情報タブ → 既存フォルダを選択 → 「コダナの使い方フォルダ」を選ぶ  

詳細は内蔵の「コダナの使い方」教材で学べます♪

## 注意点
- Windows Defenderが「不明な発行元」って警告出るかも → 「詳細情報」→「実行」を押してね
